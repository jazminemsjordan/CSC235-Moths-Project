CSC235 Moths Project

Jazmine Jordan, Krystal Bautista, Scarlett Lopez, Emma McRedmond, and Lorelei Barnum

This project was conducted using data provided by Dr. Mariana Abarca at Smith College through her moth monitoring project, referred to as "Mothitor," conducted at Smith College's Macleish Field Station. The primary research question being investigated through Mothitor is as follows: does landing surface size affect moth diversity estimates? As a team, we built to interactable plots where we utilize variable such as biomass, species richness, abundance, number of moths observed, species, date, and landing size. that address this question.

Bar Chart

Use the following bar chart to learn more about species richness, biomass, or abundance by landing size. The drop down menu below allows you to change the y-axis variable. If you hover over any of the bars it will give you a data breakdown consisting of metric, station, and value. Please note that the data shows a positive skew for the mean across all three variables. Though they vary in how extreme by variable.

Scatterplot

Use the following scatter plot to learn more about the relationship between number of moths, time of year, and landing size. Additionally, each data point is interactable! Click on a point to see a pop out of a pie chart breakdown of the number of moths by species on that date.
