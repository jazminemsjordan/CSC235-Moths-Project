CSC235 Moths Project

Jazmine Jordan, Krystal Bautista, Scarlett Lopez, Emma McRedmond, and Lorelei Barnum

This project uses data provided by Dr. Mariana Abarca at Smith College gathered through her moth monitoring project conducted at Smith College's Macleish Field Station, referred to as "Mothitor." To help in the project's research efforts to address the following research question: Does landing surface size affect moth diversity estimates? As a team, we built three interactive plots that address this question by utilizing variables such as biomass, species richness, abundance, number of moths observed, species, date, and landing size. These graphs visualize in particular the relationship between landing size and biomas/species richness/abundance, the relationship between date, number of moths, and landing size, and the relationship between date and species.

Bar Chart

Use the following bar chart to learn more about species richness, biomass, or abundance by landing size. The drop down menu below allows you to change the y-axis variable. Additionally, if you hover over any of the bars it will give you a data breakdown consisting of the metric, station, and value. Please note that the data shows a positive skew for the mean across all three variables. Though the skew varies in strength by variable.

Scatterplot

Use the following scatter plot to learn more about the relationship between number of moths, time of year, and landing size. Additionally, each data point is interactive! Click on a point to see a pop out of a pie chart breakdown of the number of moths by species on that date. Notably, there is a lack of data for the month of August. This is because the research project is not ran during August and should not be attributed to moth behavior.
